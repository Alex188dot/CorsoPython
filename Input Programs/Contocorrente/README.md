  # Checking Account

This is a simple Python input program where the user is able to withdraw money, deposit money, make a transfer to another user and view the history of their transactions. 

Demo below:

A user is able to login with their username and ID:

<img width="2062" alt="Withdraw 2" src="https://github.com/Alex188dot/CorsoPython/assets/117444853/56ee4461-152b-454e-ad68-1223f4c705b8">

By choosing 2 in the main menu, a user is able to deposit money into their account:
